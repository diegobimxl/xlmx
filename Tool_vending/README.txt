Vending Machine Analytics
=========================

Track inventory, revenue, margins, and restock needs across vending machine locations.

FEATURES
- Dashboard with KPIs: total revenue, profit, margin, avg stock level
- Machine table with location, capacity, revenue, service dates
- Product table with cost, price, margin, units sold
- Restock plan: auto-flags machines below 30% stock
- CSV export for restock routes
- Sample data for 5 machines and 8 products

DEPLOY
- Upload index.html to any subdirectory
- No dependencies, works offline
