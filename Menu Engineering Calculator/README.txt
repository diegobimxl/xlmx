Menu Engineering Calculator
===========================

Classify menu items into Stars, Plowhorses, Puzzles, and Dogs based on popularity and profitability.

FEATURES
- Auto-classifies items using contribution margin vs popularity thresholds
- KPIs: item count by category, avg contribution margin %
- Sorted matrix view by category and profit
- Add/remove items, CSV export
- Reference guide explaining each category and recommended actions
- 10 sample items pre-loaded

DEPLOY
- Upload index.html to any subdirectory
- No dependencies, works offline
