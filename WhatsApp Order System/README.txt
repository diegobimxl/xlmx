WhatsApp Order System
=====================

Catalogo de productos con carrito y generador de pedidos por WhatsApp.

FEATURES
- Product catalog grouped by category with search
- Cart with quantity +/- controls
- Generates formatted WhatsApp message with order details
- Opens wa.me link to send directly to business number
- Order history log
- Configurable business name and phone number
- Add custom products
- 12 sample products pre-loaded

DEPLOY
- Upload index.html to any subdirectory
- No dependencies, works offline
- Configure business WhatsApp number in Config tab
